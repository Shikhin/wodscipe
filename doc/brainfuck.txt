Brainfuck interpreter is mostly pretty standard one (maybe except how loops are
implemented) but does contain pretty nice debugger. To enable debugger
uncomment the third line in `lang/brainfuck.asm`.

Command  Meaning
+        *p++
-        *p--
>        p++
<        p--
.        putchar(*p);
,        c=getchar(); *p=(c>0?c:0);
[        while(*p) {
]        }
